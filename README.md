# Portafolio — Cristia Gimenez (CJ)

Portafolio personal de Cristia Gimenez, diseñador y desarrollador web.
Estética HUD cyberpunk. Sitio en producción: https://cjgimenez.com

## Stack
- HTML5 + CSS (Tailwind autocompilado)
- JavaScript vanilla
- PHP (formulario de contacto con PHPMailer)
- Fuentes locales (Space Grotesk + Material Symbols subset, sin Google Fonts)

## Características
- Diseño responsive con estética HUD cyan
- Selector de idioma persistente
- Formulario de contacto funcional con consentimiento RGPD
- Cursor personalizado y transiciones entre páginas
- Política de privacidad conforme al modelo de la AEPD
